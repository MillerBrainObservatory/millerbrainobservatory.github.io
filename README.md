## Miller Brain Observatory (MBO) Compute Hub

Source code for the [Miller Brain Observatory Computational Hub](https://millerbrainobservatory.github.io/)

To be cross-referenced, all sub-documentation (e.g. [mbo_utilities](https://millerbrainobservatory.github.io/mbo_utilities/index.html)) need the following:

- Sphinx
- MyST Parser
- Sphinx Book Theme
- Custom Extensions:
  - `sphinx_design` - Grid layouts and cards
  - `sphinx_tippy` - Tooltips for glossary terms
  - `sphinxcontrib-images` & `sphinxcontrib-video` - Media embedding
  - `sphinx-togglebutton` - Collapsible content sections
  - `numpydoc` - NumPy-style docstring rendering
  - `intersphinx` - Cross-referencing with external documentation
